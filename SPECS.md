# Tauri R2 Drive - Technical Specifications

See the complete specifications in the project documentation.

## Quick Start

1. Configure R2 credentials in Settings
2. Browse and upload files
3. Monitor transfers in real-time

## Tech Stack

- **Frontend**: Svelte 5 + TypeScript + TailwindCSS
- **Backend**: Rust + AWS SDK (S3-compatible)
- **Database**: SQLite
- **Features**: Multipart uploads, Resume capability, File sync
